# Trigger
Trigger this skill automatically whenever you are creating, refactoring, or modifying any UI components, pages, or styling views.

# Instruction
Before starting any UI task, you MUST reference the [UIContract.md](UIContract.md) file located in this same folder to ensure 100% consistency with the 'Gold Standard' view. 

Read `UIContract.md` thoroughly and apply its rules strictly to borders, padding, colors, fonts, shadow, and component structures (buttons, inputs, cards).
